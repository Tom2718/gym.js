Class DiscreteTuple

A tuple of Discrete Spaces.

A convenience class for sampling a tuple of Discrete spaces.

example
const discTuple = new DiscreteTuple([2,2,6]);
console.log(discTuple.toString());
> DiscreteTuple: [2, 2, 6]

const sample = discTuple.sample();
console.log(sample);
> [0,1,3]
